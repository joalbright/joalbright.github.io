### Information Collected

All information collected is through Apple's Game Center and Cloud Kit. Apps using Game Center will request you to sign in and do not require you to use Game Center to use the app. Apps using Cloud Kit require you to sign in using an iCloud account and do not require the account to use the app. For more information see [Apple's Privacy Policy](https://www.apple.com/legal/privacy/en-ww/)

### Technical Basics

If you enable notifications, we must store a token to send them. We never use notifications for marketing.

### Security

We implement a variety of security measures to help keep your information secure. Including using Apple's built in systems to properly secure any Game Center or Cloud Kit data. You may delete data within the respective apps, but all accounts are managed by Apple services.
